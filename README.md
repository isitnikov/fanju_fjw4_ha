# FanJu FJW4 – Home Assistant integration

Custom Home Assistant integration for FanJu FJW4 / WeatherSense weather stations.
Based on https://github.com/slavvka/homebridge-fanju-fjw4

## Features
- Indoor temperature
- Outdoor temperature
- Indoor humidity
- Outdoor humidity

## Installation (HACS)
1. HACS → Integrations → Custom repositories
2. Add this repository as **Integration**
3. Install and restart Home Assistant
4. Settings → Devices & Services → Add integration → FanJu FJW4

## Notes
This integration uses the EmaxLife cloud API.
